# Phat's Mini Games

A collection of mini games to play with friends. Built with Next.js and shadcn/ui.

## Games

- **Undercover** — social deduction game where civilians, undercover agents, and a Mr. White each receive (or don't receive) a secret word and try to figure out who's who.

- **Tiến Lên** — score tracker for the Vietnamese card game. Spreadsheet-style table with per-round entry, column totals, zero-sum validation, quick rank auto-fill, leader highlight, cumulative chart, and Slack/screenshot export.

- **Skyjo** — score tracker for Skyjo. Tracks cumulative scores across rounds with live ranking (lowest score wins), best-of-3/5/7 round counter, 100-point danger zone, and payment order (who pays first vs last).

## Running locally

```bash
bun dev
```

Open [http://localhost:2026](http://localhost:2026).
