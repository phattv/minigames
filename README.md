# Phat's Mini Games

A collection of mini games to play with friends. Built with Next.js and shadcn/ui.

## Games

- **Undercover** — social deduction game where civilians, spies, and a white hat each receive (or don't receive) a secret word and try to figure out who's who.

## Running locally

```bash
bun dev
```

Open [http://localhost:2026](http://localhost:2026).
